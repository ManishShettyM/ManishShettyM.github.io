SoftNER: Mining Knowledge Graphs From Cloud Incidents→ Manish Shetty, Chetan Bansal, Sumit Kumar, Nikitha Rao, Nachiappan Nagappan Under Submission [pdf] Extends SoftNER by transforming entities from "Strings" to "Things". Describes an unsupervised approach to mine binary entity relations from co-occurring entity pairs using pointwise mutual information and automatically construct an incident knowledge graph. Also describes some applications of SoftNER and its deployment.
Neural Knowledge Extraction From Cloud Service Incidents → Manish Shetty, Chetan Bansal, Sumit Kumar, Nikitha Rao, Nachiappan Nagappan, Thomas Zimmermann International Conference on Software Engineering (ICSE) 2021 [pdf] [talk]
Contributes SoftNER - a weak-supervised framework for named-entity extraction from software incidents. Further, the framework incorporates a novel multi-task learning based BiLSTM-CRF model which leverages not just the semantic context but also data-types for named-entity extraction.
